# Ядро интернет-магазина на Python

Базовые классы для реализации системы управления товарами и категориями в e-commerce.

## Особенности

- Класс `Product` для описания товаров
- Класс `Category` для группировки товаров
- Автоматический подсчет:
  - Общего количества категорий
  - Общего количества товаров во всех категориях
- Типизированные поля данных
- Полностью покрыто тестами

## Требования

- Python 3.7+
- Poetry (для управления зависимостями)

## Установка

1. Установите Poetry:

## Клонируйте репозиторий и установите зависимости:
poetry install 

## Использование
- Создание товаров
from main import Product

phone = Product(
    name="iPhone 15", 
    description="Флагманский смартфон",
    price=999.99,
    quantity=50
)

- Работа с категориями 
from main import Category

# Создание категории с товарами
electronics = Category(
    name="Электроника",
    description="Гаджеты и устройства",
    products=[phone, laptop]
)

print(electronics.category_count)  # Доступ к счетчику категорий
print(Category.product_count)      # Общее количество всех товаров
